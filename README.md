# Securing the Network Infrastructure

## Project Overview
This project focuses on implementing layered network security controls in a virtual enterprise environment using VirtualBox, Kali Linux, Ubuntu Server, firewalls, IDS/IPS, and traffic monitoring tools.

## Objectives
- Secure enterprise network infrastructure
- Implement firewall and IDS/IPS controls
- Simulate network attacks
- Analyze logs and alerts
- Improve network hardening

## Technologies Used
- Oracle VirtualBox
- Kali Linux
- Ubuntu Server
- UFW / iptables
- Snort IDS
- Wireshark
- Nmap
- tcpdump
